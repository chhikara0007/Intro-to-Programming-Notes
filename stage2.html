Varable Assignment

  • To introduce a variable to python you need to use an "Assignment Statement".
  • Variable names can only have letters, numbers and underscores and must begin with an underscore or letter.

example: name = expression
speed_of_light = 299792458
billionth = 1.0 / 1000000000

  • = means assignment, think of it as an arrow.
  • When you use the same variable name later in Python code with a different value the old variable will still exist but it will be overwritten with the new variable value.

Strings

A sequence of characters surrounded by single quotes or double quotes. You can 'start the string' in single quotes and "end the string" in double quotes. If a string starts with a single quote it must end in a single quote and vice vera.
ex:
print 'Hello'
print "Hello"

Concatenation

Finding Strings in Strings

Find is a method in Python.
string.find(string) = number that gives first position in search string where the target string appears. If the target is not found it will output -1.

ex:

Adding Parameters to Find

string.find(string,number) = If there are multiple occurences of your find you can decide where to start the results of your find with a number parameter.

ex:
danton = “De L’audace, encore de l’audace, toujours de l’audace."
print danton.find("audace") prints 5
print danton.find("audace", 0) prints 5
print danton.find("audace", 5) prints 5
print danton.find("audace", 6) prints 25
